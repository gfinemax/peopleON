# People On - Web Application 사용 설명서

## 1. 환경 설정 (Environment Setup)
웹 애플리케이션 실행을 위해 Supabase 연결 키 설정이 필요합니다.

1. `web/.env.local` 파일을 엽니다.
2. `NEXT_PUBLIC_SUPABASE_ANON_KEY` 값을 입력해주세요.
   - 키는 [Supabase Dashboard](https://supabase.com/dashboard) > Project Settings > API 에서 `anon` `public` 키를 복사하면 됩니다.

```bash
# web/.env.local 예시
NEXT_PUBLIC_SUPABASE_URL=https://qhmgtqihwvysfrcxelnn.supabase.co
NEXT_PUBLIC_SUPABASE_ANON_KEY=eyJhb... (여기에 붙여넣기)
```

> **주의**: `service_role` 키는 보안상 절대 `NEXT_PUBLIC_` 변수에 넣지 마세요.

## 2. 실행 방법 (Run)

터미널에서 아래 명령어로 개발 서버를 실행합니다.

```bash
cd web
npm run dev
```

브라우저에서 `http://localhost:3001`으로 접속합니다.

## 3. 구현된 기능 (Features)

### 👥 조합원 관리 (CRM)
- **대시보드**: `http://localhost:3001/members`
  - **오류 감지 기능 (권리증 충돌)**: 대시보드 진입 시 전체 조합원의 권리증 번호를 자동 분석하여, 동일한 번호를 보유한 서로 다른 명의자(중복)가 존재하면 '조치 필요 조합원' 최상단에 🚨`[충돌오류]` 펄스 배지와 함께 노출합니다. 그룹화된 리스트 우측의 '상세 보기' 클릭 시 해당 권리증 번호로 필터링된 검색 화면으로 즉시 이동하여 데이터를 병합하거나 수정할 수 있습니다.
- **조합원 검색**: 이름, 동호수, 전화번호로 실시간 검색
- **상세 페이지**: 
  - 기본 정보 (등급, 평형, 상태)
  - **타임라인**: 전화/미팅 상담 이력 기록 및 조회 (카드형 다크 UI 적용)
  - **새로운 활동 기록**: 상세 페이지에서 즉시 상담 내용 입력 및 저장 가능
  - **과거 이력**: 마이그레이션된 엑셀 데이터(`raw_data`) 원본 확인
  - **관리자 메모**: 대리인 유무와 상관없이 상세 페이지에서 항상 볼 수 있도록 상시 노출 적용
  - **데이터 정확도**: 처리된 권리증이나 잘못된 과거 내역 정보 자동 필터링 적용
- **감사 로그(Audit Logs)**: 관리자 메뉴에서 시스템 내 주요 변경, 활동내역을 요약 및 조회

### 💰 자금/권리 관리 (ERP)
- **대시보드**: `http://localhost:3001/finance`
- **통계**: 전체 과거 기록 수, 환불 대상자 수, 권리증 보유자 수, 징수 분담금 및 회차별 납부/미납 통계 현황 표출
- **권리증 통합 검수**: 등기/기타 조합원의 권리증 상태(잠정/최종), 충돌(중복), 시스템 간 미연결 데이터를 대시보드 및 상세 검색에서 통합 제공하여 누락 없는 권리증 검수를 지원합니다.
- **권리증 데이터**: 엑셀에서 추출한 권리증 보유 현황 및 원본 데이터 조회
- **조합원 상세 납부 관리**: 조회된 조합원 상세 패널의 '납부 현황' 탭을 이용해 출자금, 납부액, 계약금, 중도금, 잔금 등의 세부 항목을 입력하고 청구액 대비 과납/미납을 즉각 시각화합니다.
- **자금 기준 관리 (Settings)**: 관리자 설정 탭에서 '입금 계좌' (조합 주거래, 신탁, 임시 계좌 등) 및 '평형별 분담금 기준표'를 동적으로 관리할 수 있습니다.

### 🛡 데이터 정합성 강화 (Data Integrity)
- **중복 조합원(동명이인) 처리 로직**: 생년월일, 전화번호, 회원번호 등을 조합하여 시스템상에서 중복과 오류(생년월일이 회원번호에 저장되는 등)를 자동으로 감지하고, 이를 정상 데이터 하나로 통합하는 정렬 스크립트와 로직이 지원됩니다.
- **번호 동기화 불일치 개선**: 보유 권리증 기반 회원 대표 번호(member_number) 동기화 처리 및 `-` 출력 오류 해결.
- **대리인 권리증 합산**: 등기조합원에 연결된 대리인(agent)의 권리증이 본인 데이터에 자동 합산되어 리스트, KPI 차트, 엑셀, 인쇄에 모두 반영됩니다.

### 📥 맞춤형 데이터 출력 (Export & Print)
- **엑셀 다운로드**: 인물 리스트 화면(`/members`) 상단의 `[엑셀 다운로드]` 버튼을 클릭하면, 포함할 컬럼(성명, 연락처, 권리증번호, 구분, 동호수, 주소, 상태, 역할, 메모)을 직접 선택하는 팝업이 뜹니다. 선택 후 `[다운로드]`를 클릭하면 현재 필터링된 인원 전체가 `.xlsx` 엑셀 파일로 즉시 다운로드됩니다. **권리증번호는 화면에서 "외 N건"으로 축약되어 보여도, 엑셀에서는 모든 번호가 온전히 출력됩니다.**
- **A4 보고서 인쇄**: `[보고서 인쇄]` 버튼을 클릭하면 A4 Landscape 규격에 최적화된 깔끔한 테이블 양식이 화면에 표시되며, 브라우저의 인쇄 대화상자(`Ctrl+P`)가 자동으로 열립니다. 여기서 PDF로 저장하거나 프린터로 직접 출력할 수 있습니다.

### 📱 모바일 최적화 기능 (Adaptive UI)
- **접근 방법**: 모바일 기기로 접속 시 자동으로 전용 레이아웃으로 전환됩니다 (또는 브라우저 창 너비 축소).
- **하단 내비게이션**: 한 손 조작이 용이한 하단 고정 탭 바 (홈, 회원, 타임라인, 자금).
- **현장 특화 기능**:
  - **모바일 검색**: 대시보드 상단 검색창 터치 시, 즉시 전체 조합원 검색 팝업 실행 (이름/동호수).
  - **퀵 액션**: 회원 정보에서 즉시 전화, 문자 발송 및 상담 기록 추가 가능.
  - **풀스크린 시트**: 작은 화면에서도 정보를 명확히 볼 수 있는 전체 화면 프로필 뷰.
  - **자금 통합 뷰**: 
    - ERP 연동을 통해 전체 수납액, 미납액 통계를 실시간 그래프로 제공.
    - 회차별(1차, 2차...) 납부 상태 및 개인별 상세 미납 내역 추적 가능.

### 📱 안드로이드 앱 설치 (Native App)
- **APK 설치**: 
  1. 개발자에게 제공받은 `app-debug.apk` (또는 ZIP) 파일을 다운로드합니다.
  2. 안드로이드 폰에서 파일을 실행하여 설치합니다. (보안 설정 > '이 출처 허용' 필요)
- **특징**: 웹사이트(`people-on.vercel.app`)와 자동 동기화되며, 앱스토어 심사 없이 최신 기능을 사용할 수 있습니다.

## 4. 폴더 구조
- `src/app`: Next.js App Router 페이지
- `src/components`: UI 컴포넌트 (shadcn/ui) 및 기능별 컴포넌트
- `src/lib/supabase`: Supabase 클라이언트 설정
- `scripts/`: 데이터 마이그레이션 및 DB 설정용 파이썬/SQL 스크립트
